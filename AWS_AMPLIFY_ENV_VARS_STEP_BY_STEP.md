# AWS Amplify 환경 변수 설정 - 정확한 단계

## 🎯 현재 위치
- **페이지**: 브랜치 설정 (Branch settings)
- **보이는 것**: 브랜치 테이블, `main` 브랜치가 보라색으로 강조됨

## ✅ 정확한 클릭 순서

### 1단계: 브랜치 이름 클릭
**브랜치 테이블에서 `main` 브랜치 이름을 클릭하세요**
- 테이블의 "브랜치" 열에서 `main` 텍스트 클릭
- 또는 `main` 행 전체를 클릭

### 2단계: 브랜치 상세 페이지
클릭하면 `main` 브랜치의 상세 설정 페이지로 이동합니다.

### 3단계: 환경 변수 탭 찾기
상세 페이지에서 다음 탭들이 보일 것입니다:
- **일반** (General)
- **빌드 설정** (Build settings) 
- **환경 변수** (Environment variables) ← **여기를 클릭!**
- **리디렉션 및 재작성** (Redirects and rewrites)
- **도메인 관리** (Domain management)
- 등등

### 4단계: 환경 변수 추가
"환경 변수" 탭을 클릭하면:
- 기존 환경 변수 목록이 보임
- **"환경 변수 추가"** (Add environment variable) 버튼 클릭
- 변수 이름과 값 입력

## 🔍 만약 브랜치 이름을 클릭해도 안 되면

### 대안 1: 호스팅 메뉴에서
1. **왼쪽 사이드바** → **"호스팅"** (Hosting) 클릭
2. **`main` 브랜치 카드/링크 클릭**
3. 환경 변수 탭 찾기

### 대안 2: 작업 메뉴에서
1. **"작업"** (Actions) 드롭다운 클릭
2. **"빌드 설정 편집"** (Edit build settings) 선택
3. 빌드 설정 페이지에서 환경 변수 섹션 찾기

### 대안 3: 직접 URL 접근
브라우저 주소창에 다음 URL 입력:
```
https://console.aws.amazon.com/amplify/home?region=ap-southeast-2#/d1pjenj3kb3pau/main/environment-variables
```

## 📸 스크린샷 기준 정확한 위치

현재 보이는 화면에서:
1. **브랜치 테이블**의 **`main`** 행 찾기
2. **`main` 텍스트 클릭** (보라색 점 옆)
3. 상세 페이지로 이동
4. **"환경 변수"** 탭 클릭

## 💡 팁

### 브랜치를 클릭하는 다른 방법
- 브랜치 행 전체를 클릭
- 브랜치 이름 옆의 아이콘 클릭
- 브랜치 행의 아무 곳이나 클릭

### 환경 변수 탭이 안 보일 때
- 페이지를 아래로 스크롤
- 다른 탭들을 확인
- 브라우저 새로고침 (F5)

3. 클릭!

# AWS Amplify 환경 변수 - 정확한 위치 찾기

## 🔍 현재 상황
- 브랜치 설정 페이지에 있음
- 환경 변수 메뉴/탭이 안 보임
- URL로 직접 접속해도 앱 메인만 뜸

## ✅ 환경 변수는 여기에 있습니다!

### 방법 1: 호스팅 메뉴에서 (가장 확실한 방법)

1. **왼쪽 사이드바** → **"호스팅"** (Hosting) 클릭
   - 브랜치 설정이 아니라 **호스팅** 메뉴입니다!

2. **`main` 브랜치 카드 클릭**
   - 호스팅 페이지에 브랜치 카드들이 보일 것입니다
   - `main` 브랜치 카드를 클릭

3. **브랜치 상세 페이지에서 환경 변수 찾기**
   - 여러 탭이 보일 것입니다
   - **"환경 변수"** (Environment variables) 탭 클릭
   - 또는 **"빌드 설정"** (Build settings) 탭 → 환경 변수 섹션

### 방법 2: 빌드 설정 편집에서

1. **왼쪽 사이드바** → **"호스팅"** 클릭
2. **`main` 브랜치** 옆에 **"작업"** (Actions) 드롭다운 클릭
3. **"빌드 설정 편집"** (Edit build settings) 선택
4. 빌드 설정 페이지에서 **"환경 변수"** (Environment variables) 섹션 찾기

### 방법 3: 앱 설정에서 (앱 전체 환경 변수)

1. **왼쪽 사이드바** → **"앱 설정"** (App settings) 클릭
2. 메뉴 목록 확인:
   - 일반 설정
   - 브랜치 설정 ← 여기가 아님!
   - **환경 변수** ← 여기일 수도 있음!
   - IAM 역할

## 🎯 핵심 정리

**브랜치 설정 페이지가 아니라:**
- ✅ **"호스팅"** 메뉴로 가세요!
- ✅ 그 다음 `main` 브랜치 클릭
- ✅ 환경 변수 탭 찾기

2. **`main` 브랜치 카드/링크 클릭**

3. **환경 변수 탭 찾기**
   - 브랜치 상세 페이지에서 여러 탭 중 "환경 변수" 찾기

## 💡 왜 브랜치 설정에 없나?

- **브랜치 설정**: 브랜치 목록과 기본 설정만 관리
- **환경 변수**: 각 브랜치의 **호스팅 설정**에 포함됨
- 따라서 **"호스팅"** 메뉴에서 브랜치를 선택해야 환경 변수를 설정할 수 있습니다


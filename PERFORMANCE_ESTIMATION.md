# 성능 예측: us-central1 + gemini-2.5-pro

## 현재 환경
- **Location**: `asia-northeast1` (일본 도쿄)
- **Model**: `gemini-2.5-flash` (기본값)
- **예상 응답 시간**: 약 2-5초 (스트리밍 시작)

## 변경 후 예상 환경
- **Location**: `us-central1` (미국 아이오와)
- **Model**: `gemini-2.5-pro`

### 예상 성능 변화

#### 1. 네트워크 지연시간 (Latency)
- **asia-northeast1 → 한국**: 약 20-50ms
- **us-central1 → 한국**: 약 150-250ms
- **추가 지연**: 약 100-200ms 증가

#### 2. 모델 처리 시간
- **gemini-2.5-flash**: 빠른 응답, 약 2-5초
- **gemini-2.5-pro**: 더 정확하지만 느림, 약 5-15초
- **추가 시간**: 약 3-10초 증가

#### 3. 전체 예상 응답 시간
- **현재 (flash + asia)**: 2-5초
- **변경 후 (pro + us-central)**: 8-20초 (스트리밍 시작까지)
- **최악의 경우**: 25초까지 가능

### 권장사항
- 사용자가 한국/일본에 있다면 `asia-northeast1` 유지 권장
- 응답 속도가 중요하다면 `gemini-2.5-flash` 유지
- 정확도가 더 중요하다면 `gemini-2.5-pro` 사용 (단, 느림)

### 참고
- 실제 성능은 네트워크 상태, 서버 부하, 메시지 길이에 따라 달라질 수 있음
- 스트리밍 방식이므로 첫 토큰 도착 후 계속 받아볼 수 있음

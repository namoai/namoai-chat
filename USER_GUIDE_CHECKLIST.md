# 사용자 가이드라인 작성 체크리스트

## 필수 가이드라인 항목

### 1. 계정 관리
- [ ] **회원가입 방법**
  - 이메일/비밀번호 가입
  - Google 계정으로 가입
  - 이메일 인증 절차
  - 약관 동의 (이용약관, 개인정보처리방침, 13세 이상 동의, 보호자 동의 등)
  - 생년월일 입력 및 연령 확인
  - 닉네임 중복 확인

- [ ] **로그인 방법**
  - 이메일/비밀번호 로그인
  - Google 계정 로그인
  - 2단계 인증(2FA) 설정 및 사용
  - 로그인 실패 시 계정 잠금 안내
  - 비밀번호 찾기/재설정

- [ ] **프로필 편집 방법**
  - 닉네임 변경 (중복 확인 필요)
  - 자기소개 수정
  - 프로필 이미지 업로드/변경
  - 전화번호 변경
  - 2단계 인증 설정/해제
  - 비밀번호 변경

- [ ] **계정 관리**
  - 계정 삭제 방법
  - 로그아웃 방법

### 2. 캐릭터 관련
- [ ] **캐릭터 작성 방법**
  - 기본 정보 입력 (이름, 설명, 카테고리 등)
  - 시스템 템플릿 작성
  - 첫 상황 설정
  - 첫 메시지 작성
  - 이미지 추가 (직접 업로드 또는 AI 생성)
  - AI 이미지 생성 방법 (Replicate 사용)
  - ZIP 파일로 캐릭터 임포트
  - JSON으로 캐릭터 임포트
  - 로어북(Lorebook) 추가
  - 해시태그 설정
  - 공개/비공개 설정
  - 안전 필터 설정

- [ ] **캐릭터 탐색 및 검색**
  - 캐릭터 목록 보기
  - 태그별 필터링
  - 정렬 옵션 (최신순, 인기순, 좋아요순)
  - 검색 기능 사용법
  - 캐릭터 상세 정보 보기
  - 좋아요/즐겨찾기 기능

- [ ] **캐릭터 관리**
  - 내가 만든 캐릭터 목록 보기
  - 캐릭터 수정
  - 캐릭터 삭제

### 3. 채팅 관련
- [ ] **채팅 시작하기**
  - 새 채팅 시작 방법
  - 페르소나 선택 (채팅 시 사용할 역할 설정)
  - 채팅 설정 (모델 선택, 온도 등)

- [ ] **채팅 방법**
  - 메시지 전송
  - 메시지 재생성 (Regenerate)
  - 응답 버전 선택 (여러 버전 중 선택)
  - 채팅 내역 확인
  - 채팅 삭제

- [ ] **채팅 목록 관리**
  - 채팅 목록 보기
  - 채팅 검색
  - 채팅 선택 삭제
  - 채팅 정렬

### 4. 포인트 및 결제
- [ ] **포인트 확인**
  - 현재 포인트 잔액 확인 (무료 포인트/유료 포인트)
  - 포인트 사용 내역

- [ ] **충전 방법**
  - 포인트 패키지 선택
  - Stripe를 통한 결제 절차
  - 결제 성공/실패 처리
  - 결제 완료 후 포인트 적립 확인

- [ ] **포인트 획득**
  - 출석 체크 (1일 1회, 30포인트)
  - 포인트 사용 (채팅 1회당 소모, 이미지 생성당 소모)

### 5. 페르소나 관리
- [ ] **페르소나 작성**
  - 페르소나란 무엇인가 (설명)
  - 닉네임, 나이, 성별, 설명 입력
  - 기본 페르소나 설정

- [ ] **페르소나 관리**
  - 페르소나 목록 보기
  - 페르소나 수정
  - 페르소나 삭제
  - 기본 페르소나 변경

### 6. 이미지 생성
- [ ] **AI 이미지 생성 방법**
  - 이미지 생성 기능 위치 (캐릭터 작성 시)
  - 프롬프트 작성 방법
  - 이미지 스타일 선택 (2D/애니메만 지원)
  - 생성 시간 및 비용 안내
  - 생성된 이미지 확인 및 사용

## 추가 가이드라인 항목 (선택사항)

### 7. 고급 기능
- [ ] **안전 필터 설정**
  - 안전 필터란 무엇인가
  - 안전 필터 켜기/끄기
  - 미성년자 자동 적용 안내

- [ ] **차단 기능**
  - 사용자 차단 방법
  - 차단 해제 방법
  - 차단된 사용자의 캐릭터 숨김 처리

- [ ] **알림 및 공지사항**
  - 공지사항 확인
  - 알림 설정

### 8. 문제 해결
- [ ] **자주 묻는 질문 (FAQ)**
  - 로그인이 안 될 때
  - 포인트가 차감되지 않을 때
  - 이미지 생성이 실패할 때
  - 채팅이 느릴 때

- [ ] **문의하기**
  - 문의 방법 안내
  - 버그 신고 방법

### 9. 기타
- [ ] **랭킹 시스템**
  - 인기 캐릭터 랭킹 보기

- [ ] **프로필 공개**
  - 다른 사용자 프로필 보기
  - 팔로우 기능 (있는 경우)

## 가이드라인 작성 시 고려사항

1. **스크린샷 포함**: 각 단계마다 스크린샷을 포함하면 이해하기 쉬움
2. **단계별 설명**: 명확한 번호 매기기와 단계별 설명
3. **주의사항 강조**: 중요한 주의사항은 별도로 강조
4. **용어 설명**: 전문 용어나 특수 기능은 용어 설명 포함
5. **다국어 지원**: 한국어, 일본어 등 필요한 언어로 작성
6. **최신 정보**: UI 변경 시 가이드라인도 업데이트 필요

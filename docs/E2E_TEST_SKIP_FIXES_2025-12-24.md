## 주요 수정 사항 요약

### 선택자 개선
1. **출석 체크 버튼**: `button:has-text("出席して30ポイントGET")`
2. **좋아요 버튼**: `button:has(svg)` + 클래스 확인
3. **댓글 입력란**: `input[type="text"][placeholder*="コメントを追加"]`
4. **공지사항**: `button` 선택자 사용
5. **공지사항 내용**: `main, article, [class*="content"]`

### 로직 개선
1. **신고 기능**: 케밥 메뉴 열기 → "通報する" 클릭
2. **팔로우 기능**: 캐릭터 작성자 프로필로 이동
3. **패스워드 변경**: 프로필 페이지 케밥 메뉴에서 접근

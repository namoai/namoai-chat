## 1. 유저 관점 시나리오

### 1.1 인증·계정 관리

#### 시나리오 1-1-1: 메일 주소 회원가입
- 회원가입 페이지 접근
- 메일 주소, 비밀번호, 이름, 전화번호, 닉네임 입력
- 회원가입 실행
- 프로필 완성 페이지로 리다이렉트 확인
- 프로필 완성 (생년월일, 성별, 자기소개 등 입력)
- 완료 후 홈 화면으로 이동 확인

#### 시나리오 1-1-2: Google 계정 회원가입
- Google 로그인 버튼 클릭
- Google 인증 진행
- 추가 정보 입력 (닉네임 등)
- 회원가입 완료 확인

#### 시나리오 1-1-3: 메일 주소 로그인
- 로그인 페이지 접근
- 메일 주소와 비밀번호 입력
- 로그인 버튼 클릭
- 홈 화면으로 리다이렉트 확인
- 세션 유지 확인

#### 시나리오 1-1-4: Google 계정 로그인
- Google 로그인 버튼 클릭
- Google 인증 진행
- 로그인 완료 후 홈 화면 이동 확인

#### 시나리오 1-1-5: 로그아웃
- 로그아웃 버튼 클릭
- 로그아웃 완료 확인
- 로그인 페이지로 리다이렉트 확인

#### 시나리오 1-1-6: 비밀번호 변경
- 마이페이지 접근
- 비밀번호 변경 메뉴 선택
- 현재 비밀번호 입력
- 새 비밀번호 입력 및 확인
- 변경 실행
- 변경 성공 메시지 확인
- 새 비밀번호로 로그인 확인

#### 시나리오 1-1-7: 2FA (2단계 인증) 활성화
- 2FA 설정 페이지 접근
- 2FA 활성화 실행
- 인증 코드 입력
- 활성화 완료 확인
- 다음 로그인 시 2FA 코드 요청 확인

#### 시나리오 1-1-8: 이메일 인증
- 이메일 인증 요청
- 인증 메일 수신 확인
- 인증 링크 클릭
- 인증 완료 확인

### 1.2 캐릭터 탐색·검색

#### 시나리오 1-2-1: 캐릭터 검색
- 홈 화면 또는 검색 페이지 접근
- 검색창에 키워드 입력
- 검색 실행
- 검색 결과 목록 표시 확인
- 검색 결과에서 캐릭터 선택
- 캐릭터 상세 페이지 이동 확인

#### 시나리오 1-2-2: 캐릭터 일람 확인
- 캐릭터 일람 페이지 접근
- 캐릭터 목록 표시 확인
- 카테고리 필터로 필터링 (예: "로맨스", "판타지" 등)
- 태그 필터로 필터링
- 정렬 기능 확인 (신규순, 인기순, 좋아요순)
- 페이지네이션 동작 확인

#### 시나리오 1-2-3: 캐릭터 상세 확인
- 캐릭터 상세 페이지 접근
- 캐릭터 이름, 설명 표시 확인
- 캐릭터 이미지 표시 및 이미지 전환 확인
- 상세 설정, 첫 상황, 첫 메시지 표시 확인
- 좋아요 수, 댓글 수 표시 확인
- 제작자 정보 표시 확인

#### 시나리오 1-2-4: 블록한 제작자의 캐릭터 검색 제외 확인
- 타 사용자 프로필 접근
- 사용자 블록 실행
- 캐릭터 검색 실행
- 블록한 사용자의 캐릭터가 검색 결과에 나타나지 않는지 확인
- 캐릭터 일람에서도 제외되는지 확인
- 랭킹에서도 제외되는지 확인

---

### 1.3 캐릭터 작성·관리

#### 시나리오 1-3-1: 캐릭터 작성 (AI 생성 기능 사용)
- 캐릭터 작성 페이지 접근
- 카테고리 선택
- AI 프로필 생성 버튼 클릭
- 생성된 이름, 설명 확인
- AI 상세 설정 생성 버튼 클릭
- 생성된 상세 설정 확인
- AI 첫 상황 생성 버튼 클릭
- 생성된 첫 상황, 첫 메시지 확인
- 이미지 업로드 (여러 장)
- 메인 이미지 설정
- 해시태그 추가
- 공개 설정 선택 (공개/비공개)
- 세이프티 필터 설정
- 저장 실행
- 작성 완료 확인
- 작성한 캐릭터가 내 캐릭터 목록에 표시되는지 확인

#### 시나리오 1-3-2: 캐릭터 작성 (수동 입력)
- 캐릭터 작성 페이지 접근
- 이름, 설명, 상세 설정, 첫 상황, 첫 메시지 수동 입력
- 이미지 업로드
- 카테고리, 해시태그 설정
- 공개 설정, 세이프티 필터 설정
- 저장 실행
- 작성 완료 확인

#### 시나리오 1-3-3: 캐릭터 편집
- 내 캐릭터 관리 페이지 접근
- 편집할 캐릭터 선택
- 편집 페이지로 이동
- 정보 수정 (이름, 설명 등)
- 이미지 추가/삭제
- 저장 실행
- 변경 사항이 반영되었는지 확인
- 캐릭터 상세 페이지에서 변경 확인

#### 시나리오 1-3-4: 캐릭터 삭제
- 내 캐릭터 관리 페이지 접근
- 삭제할 캐릭터 선택
- 삭제 버튼 클릭
- 확인 다이얼로그에서 확인
- 삭제 완료 확인
- 캐릭터 일람에서 삭제된 캐릭터가 보이지 않는지 확인

#### 시나리오 1-3-5: 캐릭터 클론
- 타 사용자가 작성한 캐릭터 상세 페이지 접근
- 클론 버튼 클릭
- 클론된 캐릭터가 내 캐릭터 목록에 추가되는지 확인
- 클론된 캐릭터 편집 가능 확인

#### 시나리오 1-3-6: 캐릭터 Export
- 내 캐릭터 관리 페이지 접근
- Export할 캐릭터 선택
- Export 버튼 클릭
- JSON 파일 다운로드 확인

#### 시나리오 1-3-7: 캐릭터 Import
- 캐릭터 작성 페이지 접근
- Import 버튼 클릭
- JSON 파일 선택
- 데이터 로드 확인
- 편집 후 저장 가능 확인

### 1.4 채팅 기능

#### 시나리오 1-4-1: 채팅 시작 (신규)
- 캐릭터 상세 페이지 접근
- "채팅 시작" 버튼 클릭
- 채팅 페이지로 이동 확인
- 첫 메시지 표시 확인
- 채팅 ID 생성 확인

#### 시나리오 1-4-2: 메시지 전송·수신
- 채팅 화면에서 메시지 입력
- 전송 버튼 클릭
- 내 메시지가 화면에 표시되는지 확인
- 포인트 차감 확인 (필요시)
- AI 응답 대기 (로딩 표시 확인)
- AI 응답 수신 확인
- 응답이 화면에 표시되는지 확인

#### 시나리오 1-4-3: 채팅 이력 확인
- 채팅 목록 페이지 접근
- 과거 채팅 목록 표시 확인
- 특정 채팅 선택
- 채팅 내용 불러오기 확인
- 메시지 이력 표시 확인

#### 시나리오 1-4-4: 채팅 취소
- 채팅 중인 상태에서 취소 버튼 클릭
- 취소 확인 다이얼로그 표시
- 취소 실행
- 채팅이 취소되었는지 확인
- 포인트 환불 확인 (필요시)

#### 시나리오 1-4-5: 채팅 노트 작성
- 채팅 화면에서 노트 기능 사용
- 노트 작성
- 저장 확인
- 다음 접속 시 노트 유지 확인

#### 시나리오 1-4-6: 세이프티 필터 제약 확인
- 세이프티 필터 ON 상태 확인
- 세이프티 필터 OFF 캐릭터 상세 페이지 접근
- "채팅 시작" 버튼 클릭 시도
- 에러 메시지 표시 확인 (채팅 시작 불가)
- 세이프티 필터 OFF로 변경
- 다시 채팅 시작 시도
- 채팅 시작 가능 확인

#### 시나리오 1-4-7: 채팅 랭킹 반영 확인
- 캐릭터와 채팅 시작
- 여러 메시지 교환 (최소 5개 이상)
- 랭킹 페이지 접근
- 해당 캐릭터가 랭킹에 반영되어 있는지 확인
- 채팅 수가 정확히 집계되었는지 확인
- 실시간/일일/주간/월간 랭킹 모두 확인

### 1.5 포인트·결제 기능

#### 시나리오 1-5-1: 포인트 잔액 확인
- 포인트 페이지 접근
- 무료 포인트, 유료 포인트 표시 확인
- 총 포인트 표시 확인

#### 시나리오 1-5-2: 포인트 충전
- 포인트 페이지 접근
- 충전 버튼 클릭
- 금액 선택 또는 입력
- 결제 진행 (Stripe 연동)
- 결제 완료 확인
- 포인트 페이지로 리다이렉트 확인
- 충전된 포인트가 잔액에 반영되었는지 확인

#### 시나리오 1-5-3: 출석 체크
- 포인트 페이지 접근
- 출석 체크 버튼 클릭
- 포인트 지급 확인 (30 포인트)
- 출석 체크 완료 메시지 확인
- 다음 날 출석 체크 시도
- "이미 출석 체크 완료" 메시지 확인 (또는 정상 지급)

#### 시나리오 1-5-4: 포인트 사용 확인
- 채팅 메시지 전송
- 포인트 차감 확인
- 포인트 페이지에서 잔액 확인
- 사용 이력 확인 (가능한 경우)

### 1.6 소셜 기능

#### 시나리오 1-6-1: 자신의 프로필 확인
- 마이페이지 또는 프로필 페이지 접근
- 프로필 정보 표시 확인 (닉네임, 자기소개 등)
- 내가 작성한 캐릭터 목록 표시 확인
- 팔로워 수, 팔로잉 수 표시 확인
- 총 채팅 메시지 수 표시 확인

#### 시나리오 1-6-2: 타 사용자 프로필 확인
- 타 사용자 프로필 페이지 접근
- 프로필 정보 표시 확인
- 해당 사용자가 작성한 캐릭터 목록 표시 확인
- 팔로우 버튼 표시 확인
- 블록 버튼 표시 확인 (있는 경우)

#### 시나리오 1-6-3: 팔로우/언팔로우
- 타 사용자 프로필 페이지 접근
- 팔로우 버튼 클릭
- 팔로우 상태로 변경 확인
- 팔로잉 수 증가 확인
- 해당 사용자의 팔로워 수 증가 확인
- 다시 팔로우 버튼 클릭 (언팔로우)
- 언팔로우 확인
- 팔로워/팔로잉 수 감소 확인

#### 시나리오 1-6-4: 팔로워/팔로잉 목록 확인
- 프로필 페이지 접근
- 팔로워 수 또는 팔로잉 수 클릭
- 팔로워 목록 표시 확인
- 팔로잉 목록 표시 확인
- 목록에서 사용자 클릭 시 해당 사용자 프로필로 이동 확인

#### 시나리오 1-6-5: 사용자 블록/언블록
- 타 사용자 프로필 페이지 접근
- 블록 버튼 클릭
- 블록 확인 다이얼로그 표시
- 블록 실행
- 블록 완료 확인
- 블록한 사용자 목록에서 확인
- 블록한 사용자의 캐릭터가 검색 결과에서 제외되는지 확인
- 언블록 실행
- 언블록 후 다시 검색 결과에 나타나는지 확인

#### 시나리오 1-6-6: 블록한 사용자 목록 확인
- 마이페이지 또는 설정 페이지 접근
- 블록한 사용자 목록 메뉴 선택
- 블록한 사용자 목록 표시 확인
- 언블록 기능 동작 확인

#### 시나리오 1-6-7: 좋아요 기능
- 캐릭터 상세 페이지 접근
- 좋아요 버튼 클릭
- 좋아요 상태로 변경 확인
- 좋아요 수 증가 확인
- 다시 좋아요 버튼 클릭 (좋아요 해제)
- 좋아요 해제 확인
- 좋아요 수 감소 확인

#### 시나리오 1-6-8: 댓글 작성
- 캐릭터 상세 페이지 접근
- 댓글 입력란에 댓글 작성
- 댓글 작성 버튼 클릭
- 댓글이 댓글 목록에 추가되는지 확인
- 작성자 정보, 작성 시간 표시 확인

#### 시나리오 1-6-9: 댓글 삭제
- 자신이 작성한 댓글이 있는 캐릭터 상세 페이지 접근
- 댓글 삭제 버튼 클릭
- 삭제 확인 다이얼로그 표시
- 삭제 실행
- 댓글이 목록에서 제거되는지 확인

#### 시나리오 1-6-10: 댓글 알림 확인
- 타 사용자가 내 캐릭터에 댓글 작성
- 알림 페이지 접근
- 댓글 알림 표시 확인
- 알림 클릭 시 해당 캐릭터 상세 페이지로 이동 확인

### 1.7 알림 기능

#### 시나리오 1-7-1: 알림 일람 확인
- 알림 페이지 접근
- 알림 목록 표시 확인
- 알림 종류별 표시 확인 (팔로우, 좋아요, 댓글 등)
- 알림 시간순 정렬 확인

#### 시나리오 3-14-2: 이름/설명 검색
- 검색창에 캐릭터 이름 또는 설명 키워드 입력
- 검색 실행
- 관련 캐릭터 표시 확인

#### 시나리오 3-14-3: 검색 결과 정렬
- 검색 결과 확인
- 정렬 옵션 확인 (인기순, 신규순 등)
- 정렬 변경 시 결과 재정렬 확인

### 3.15 관리자 기능 세부 사항

#### 시나리오 3-15-1: 사용자 포인트 수동 설정
- 사용자 편집 모달 열기
- 무료 포인트 수동 입력
- 유료 포인트 수동 입력
- 저장 실행
- 사용자 포인트 페이지에서 설정된 포인트 확인

#### 시나리오 3-15-2: 나모아이 프렌즈 등록
- 캐릭터 관리 페이지 접근
- 캐릭터를 나모아이 프렌즈로 등록
- 홈 화면에서 나모아이 프렌즈 섹션에 표시 확인

#### 시나리오 3-15-3: 공지사항 공개/비공개
- 공지사항 작성 시 공개 설정
- 공개로 설정한 공지사항이 일반 사용자 화면에 표시되는지 확인
- 비공개로 설정한 공지사항이 일반 사용자 화면에 표시되지 않는지 확인

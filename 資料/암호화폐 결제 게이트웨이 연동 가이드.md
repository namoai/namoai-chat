# 암호화폐 결제 게이트웨이 연동 가이드

## 개요
기존 서비스에 암호화폐 결제 옵션을 추가하는 것 - **서드파티 결제 게이트웨이 API 연동**

## 1. 주요 암호화폐 결제 게이트웨이 서비스

### 1.1 CoinPayments
- **특징**
  - 2,000+ 암호화폐 지원
  - 구독/반복 결제 기능
  - IPN (Instant Payment Notification)
  - API 문서 잘 되어있음
  
- **수수료**
  - 거래당 0.5%
  - 최소 수수료 없음
  
- **연동 난이도**: ⭐⭐ (쉬움)
- **공식 사이트**: https://www.coinpayments.net/
- **성인 콘텐츠**: ✅ 허용

### 1.2 NOWPayments
- **특징**
  - 150+ 암호화폐 지원
  - 자동 환전 기능 (암호화폐 → 법정화폐)
  - 간단한 API
  - 정기 결제 지원
  
- **수수료**
  - 거래당 0.5% - 1%
  - KYC 레벨에 따라 차등
  
- **연동 난이도**: ⭐ (매우 쉬움)
- **공식 사이트**: https://nowpayments.io/
- **성인 콘텐츠**: ✅ 허용 (확인 필요)

### 1.3 BTCPay Server
- **특징**
  - 완전 무료 (오픈소스)
  - 수수료 없음 (네트워크 수수료만)
  - 자체 호스팅 필요
  - 중개자 없음 → 완전한 통제권
  
- **수수료**
  - 0% (서버 운영비용만)
  
- **연동 난이도**: ⭐⭐⭐⭐ (복잡 - 서버 구축 필요)
- **공식 사이트**: https://btcpayserver.org/
- **성인 콘텐츠**: ✅ 허용 (자체 운영)

### 1.4 CoinGate
- **특징**
  - 70+ 암호화폐 지원
  - Lightning Network 지원
  - 유럽 기반
  - 자동 환전 가능
  
- **수수료**
  - 거래당 1%
  
- **연동 난이도**: ⭐⭐ (쉬움)
- **공식 사이트**: https://coingate.com/
- **성인 콘텐츠**: ⚠️ 확인 필요

### 1.5 Coinbase Commerce
- **특징**
  - 신뢰도 높음 (Coinbase 운영)
  - 주요 암호화폐만 지원 (BTC, ETH, USDC 등)
  - 직관적인 대시보드
  
- **수수료**
  - 거래당 1%
  
- **연동 난이도**: ⭐ (매우 쉬움)

## 3. 연동 방법 (NOWPayments 기준)

### 3.1 계정 생성
1. NOWPayments 웹사이트 방문
2. 회원가입
3. KYC 진행 (간단 - 사업자 정보 제출)
4. API Key 발급

### 3.2 API 연동 (기본 플로우)

```javascript
// 1. 결제 생성 (백엔드)
const createPayment = async (amount, orderId) => {
  const response = await fetch('https://api.nowpayments.io/v1/payment', {
    method: 'POST',
    headers: {
      'x-api-key': 'YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      price_amount: amount,
      price_currency: 'usd',
      pay_currency: 'btc', // 또는 사용자가 선택한 암호화폐
      order_id: orderId,
      order_description: '챗봇 구독권',
      ipn_callback_url: 'https://yoursite.com/webhook/nowpayments'
    })
  });
  
  return await response.json();
};

// 2. 결제 URL로 리디렉션
// response.invoice_url로 사용자를 보냄

// 3. Webhook으로 결제 상태 수신 (백엔드)
app.post('/webhook/nowpayments', (req, res) => {
  const { payment_status, order_id } = req.body;
  
  if (payment_status === 'finished') {
    // 결제 완료 처리
    activateSubscription(order_id);
  }
  
  res.status(200).send('OK');
});
```

### 4.2 비용
- **개발 비용**: 거의 없음 (기존 개발자가 작업)
- **계정 등록 비용**: 무료
- **월 구독료**: 무료 (거래당 수수료만)

**초기 비용: $0 ~ 거의 없음**

### 4.3 KYC
- **판매자(당신)**: 간단한 사업자 KYC만 (게이트웨이에)
  - 사업자 정보
  - 이메일 인증
  - 신원 확인 (신분증)
  
- **구매자(사용자)**: KYC 불필요!
  - 암호화폐 지갑만 있으면 됨
  - 익명 결제 가능

### 4.4 일본 규제
- **당신(판매자) 입장**: 
  - ✅ 결제 게이트웨이가 라이선스 보유
  - ✅ 당신은 암호화폐 교환업자가 아님
  - ✅ 단순히 결제 수단을 제공하는 것뿐
  - ⚠️ 세금 신고는 해야 함 (일반 매출로)

## 6. 구현 예시 (Node.js + Express)

### 6.1 백엔드 (server.js)

```javascript
const express = require('express');
const axios = require('axios');
const app = express();

const NOWPAYMENTS_API_KEY = process.env.NOWPAYMENTS_API_KEY;
const NOWPAYMENTS_IPN_SECRET = process.env.NOWPAYMENTS_IPN_SECRET;

// 결제 생성 엔드포인트
app.post('/api/payment/create', async (req, res) => {
  const { userId, planId, amount } = req.body;
  
  try {
    const payment = await axios.post(
      'https://api.nowpayments.io/v1/payment',
      {
        price_amount: amount,
        price_currency: 'usd',
        pay_currency: 'btc', // 사용자가 선택하게 할 수도 있음
        order_id: `user_${userId}_plan_${planId}_${Date.now()}`,
        order_description: '나모스 챗봇 구독권',
        ipn_callback_url: 'https://yoursite.com/api/payment/webhook',
        success_url: 'https://yoursite.com/payment/success',
        cancel_url: 'https://yoursite.com/payment/cancel'
      },
      {
        headers: {
          'x-api-key': NOWPAYMENTS_API_KEY,
          'Content-Type': 'application/json'
        }
      }
    );
    
    res.json({
      paymentUrl: payment.data.invoice_url,
      paymentId: payment.data.payment_id
    });
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});

// Webhook 엔드포인트 (결제 완료 알림 수신)
app.post('/api/payment/webhook', express.raw({type: 'application/json'}), async (req, res) => {
  const payload = req.body;
  
  // 서명 검증 (보안)
  const signature = req.headers['x-nowpayments-sig'];
  // ... 서명 검증 로직
  
  const { payment_status, order_id, payment_id } = JSON.parse(payload);
  
  if (payment_status === 'finished') {
    // 결제 완료! 구독 활성화
    const userId = extractUserIdFromOrderId(order_id);
    await activateUserSubscription(userId);
    
    console.log(`✅ 결제 완료: ${order_id}`);
  } else if (payment_status === 'failed') {
    console.log(`❌ 결제 실패: ${order_id}`);
  }
  
  res.status(200).send('OK');
});

app.listen(3000);
```

    setLoading(true);

    }

      onClick={handlePayment} 
      disabled={loading}
      className="crypto-payment-btn"
    >
      {loading ? '처리 중...' : '암호화폐로 결제하기 💰'}
    </button>
  );
};

export default CryptoPaymentButton;

## 8. 가격 변동성 대응

### 8.1 문제
암호화폐는 가격 변동이 심함 → 받는 금액이 달라질 수 있음

### 8.2 해결책

#### 옵션 1: 스테이블코인 위주 (추천)
```javascript
// USDT, USDC만 받기
const acceptedCoins = ['usdt', 'usdc'];
```

#### 옵션 2: 즉시 환전
```javascript
// NOWPayments 설정에서 활성화
{
  "auto_conversion": true,
  "target_currency": "usd" // 또는 jpy
}
```
→ 암호화폐로 받아서 즉시 법정화폐로 환전

#### 옵션 3: 가격 고정 시간 설정
```javascript
{
  "fixed_rate": true,  // 가격 고정
  "lock_time": 900     // 15분간 고정
}
```
